Welcome to my blog!

This is a simple blog built using Django and Bootstrap. Admin and Staff members can create, edit, delete and publish posts through the 'Blog' tab. Registered users and anonymous users can also leave comments in each post.


There is also a contact form (which doesn't post data anywhere) and a navbar with Search functionality. All blog posts can be viewed in the 'Blog' tab. In this home page, only the latest post is showed with a brief description.
